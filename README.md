#Dragon Speech Kit Demo App


Continously Listen


Getting Started
====

Make sure you have XCode on your Mac.

Go to XCode, build, and run (demo is on iPhone 6 simulator).




Product Overview
====

Press "Record."

The program listens continously without stopping.
